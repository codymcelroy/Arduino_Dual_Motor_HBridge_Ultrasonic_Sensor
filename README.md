# Arduino Dual Motor H-Bridge Ultrasonic Sensor

A simple robot design with 2 motors being controlled by a distance sensor





## Bill of Materials
* (1)	Arduino
* (1) H-bridge Motor Drive **[UL298N/L293D]**
* (1) 5 Power Supply 
* (2) Hobby Gearmotor
* (1) Ultrasonic Distance Sensor **[HC-SR04]**
